# Bongure Client

**Bongure Client** is a fork of **Rushie Client (https://github.com/Rushie-Client/RushieClient-ddnet/)** with a my little functions.

## Installation

For installation **Bongure Client** download a zip from **(https://github.com/Cser228/BongureClient/Releases)** and unzip this.

## Build

For build download Visual Studio, open src folder and visual studio will find CMakeLists.txt and you will need press button run for build

## Features

* **Rushie Client**: full
* **My functions**: chat command /unfinishsay, voice assistent bonga, f1 command bonga_voice
